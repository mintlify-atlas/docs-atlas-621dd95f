## Prerequisites

Before you begin, make sure you have the following installed and available:

- **Node.js 18+**
- **PostgreSQL 14+** running locally or accessible remotely
- **Wompi sandbox credentials** — sign up at [sandbox.wompi.co](https://sandbox.wompi.co) to get your public key, private key, events key, and integrity key

<Note>
  All steps below use the Wompi sandbox environment (`api-sandbox.co.uat.wompi.dev/v1`). No real payments are processed.
</Note>

## Setup

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/jhonarodriguez/payment-checkout-app.git
    cd payment-checkout-app
    ```
  </Step>

  <Step title="Start the backend">
    ```bash
    cd backend
    npm install
    cp .env.example .env
    ```

    Open `.env` and fill in the required values:

    ```bash
    # Database
    DB_HOST=localhost
    DB_PORT=5432
    DB_USER=your_db_user
    DB_PASS=your_db_password
    DB_NAME=checkout_db

    # Wompi credentials (from your sandbox dashboard)
    PAYMENT_PUBLIC_KEY=pub_stagtest_...
    PAYMENT_PRIVATE_KEY=prv_stagtest_...
    PAYMENT_EVENTS_KEY=stagtest_events_...
    PAYMENT_INTEGRITY_KEY=stagtest_integrity_...
    PAYMENT_API_URL=https://api-sandbox.co.uat.wompi.dev/v1

    # Server
    PORT=3001
    FRONTEND_URL=http://localhost:5173

    # Fees (in cents COP)
    BASE_FEE_IN_CENTS=300000
    DELIVERY_FEE_IN_CENTS=200000
    ```

    Then start the development server:

    ```bash
    npm run start:dev
    ```

    The API is available at `http://localhost:3001/api`. Swagger docs are at `http://localhost:3001/api/docs`.

    On first boot the backend seeds the database with 3 test products if the table is empty:

    | Product | Price |
    |---------|-------|
    | Audífonos Bluetooth Pro | $150.000 COP |
    | Teclado Mecánico RGB | $250.000 COP |
    | Mouse Inalámbrico Ergonómico | $80.000 COP |
  </Step>

  <Step title="Start the frontend">
    Open a new terminal window, then:

    ```bash
    cd frontend
    npm install
    cp .env.example .env
    ```

    Open `.env` and fill in your Wompi public key:

    ```bash
    VITE_API_URL=http://localhost:3001/api
    VITE_PAYMENT_API_URL=https://api-sandbox.co.uat.wompi.dev/v1
    VITE_PAYMENT_PUBLIC_KEY=pub_stagtest_...
    ```

    Then start the dev server:

    ```bash
    npm run dev
    ```

    The app is available at `http://localhost:5173`.
  </Step>

  <Step title="Try a test purchase">
    Open [http://localhost:5173](http://localhost:5173) in your browser.

    1. Select one of the three seeded products.
    2. Fill in delivery details and a test card number. Wompi sandbox accepts any valid Luhn number — for example, `4111111111111111` (Visa).
    3. Review the order summary and confirm.
    4. Wait for the payment result. The app polls the transaction status every 2.5 seconds for up to 30 seconds.
  </Step>
</Steps>
